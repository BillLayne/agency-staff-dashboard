# DL-123 Generator

Standalone version of the Bill Layne Insurance `DL123` document generator.

## What it does

- Generates a printable NC `DL-123 Driver License Liability Insurance Certification`
- Auto-fills known agency codes for supported carriers
- Auto-calculates the expiration date as one year after the effective date
- Saves and reloads form data in browser storage
- Prints, downloads, or copies the generated document for Gmail

## Run it

Open [index.html](./index.html) in a browser.

No build step is required.
