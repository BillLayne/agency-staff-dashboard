# Auto ID Card Maker

Standalone version of the Bill Layne Insurance Auto ID Card Maker from the older document generator.

## What it does

- Generates printable North Carolina auto insurance ID cards
- Supports 1 to 4 vehicles
- Auto-fills NAIC numbers by carrier
- Auto-fills expiration date one year after the effective date
- Saves and reloads form data in browser storage
- Prints, downloads, or copies the generated card sheet for Gmail

## Run it

Open [index.html](./index.html) in a browser.

No build step is required.
